# Hospital Booking System
A Python-based clinic management system that allows staff and clients to manage appointments through a command-line interface using object-oriented programming.

## Features
- Add, edit, and delete staff members (GP, Nurse, Psychologist)
- Book, edit, and cancel appointments
- Automatic assignment of available staff based on workload
- Input validation for dates, times, and user details
- Role-based menus (Staff Menu and Client Menu)
- Save and load appointment data from text files

## Technologies Used
- Python
- Object-Oriented Programming (OOP)

## Concepts Used
- Class design for system components (BookingSystem, Client, Staff)
- Encapsulation and data management
- Polymorphism for different appointment types
